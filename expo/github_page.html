GitHub Pages로 웹 페이지 만들기

  • 이전 글 ( GitHub Repository에 파일 업로드하기 )을 먼저 읽고 따라해봅니다.
  • Repository에 index.html이 있어야 합니다.

github.com에 있는 본인의 정보문화기술입문 관련 Repoitory에 들어갑니다.

Settings > Pages를 선택합니다.

Branch를 None에서 main으로 변경합니다.

Save버튼을 누릅니다.

새로 고침을 계속하면서 기다리다 보면(5분 이내) 링크가 생성되는 것을 확인할 수 있습니다.
