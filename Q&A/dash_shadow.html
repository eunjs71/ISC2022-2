캐릭터가 커지고 작아지는 극적인 효과 만들기

  • 예시 코드 : https://github.com/eunjs71/ISC2022-2/tree/main/examples/dash_shadow
  • 예시 스케치 보기

캐릭터가 2가지 모드가 있고, 두 모드를 변경할 때, 모드를 변경하는 함수를 호출하는 동시에 각 모드에 맞는 이미지를 화면에 그리게 되면, 커지거나, 작아지는 것처럼 보이지 않고, 순간적으로 반짝 나타나게 됩니다. 이동하는 두 위치 사이에 잔상 효과를 주게 되면, 마치 커지면서 뛰어 나오거나, 작아지면서 다시 돌아오는 애니메이션을 구현할 수 있습니다.

달려나가면서 커지고, 다시 돌아오면 줄어드는 Jake라는 객체가 있습니다. Jake는 4가지 상태가 있습니다.

  • SMALL : 작은 상태
  • GET_BIGGER : 작은 상태 → 큰 상태
  • BIG : 큰 상태
  • GET_SMALLER : 큰 상태 → 작은 상태

Jake의 상태가 SMALL, 혹은 BIG일 때, form_change() 함수를 호출하면, 각각 GET_BIGGER, 혹은 GET_SMALLER 상태가 됩니다. 작은 상태, 큰 상태의 Jake의 좌표 및 크기 ( x, y, w, h )를 통해 이동하는 중간 지점들의 좌표 및 크기를 map 함수를 통해 구할 수 있습니다. 이동하면서 Shadow 객체를 생성하며, 이 객체는 시간이 지날 수록 투명해지다 사라집니다.

아래 코드는 setup() 함수에서 Jake 객체를 생성하는 부분입니다. 


            ...

            function setup() {
                createCanvas(600, 600);
              
                jake = new Jake(
                  small_jake_info,
                  big_jake_info,
                  img_big_jake,
                  200, // speed
                  10 //dim
                );
              
                textAlign(LEFT, BOTTOM);
              }
              ...

4번째 패러미터는 이동 속도, 5번째 패러미터는 그림자가 투명해지는 속도입니다. 조절해가며 잔상의 정도 및 효과를 조절해보세요.
