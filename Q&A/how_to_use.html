[HTML, CSS] 스케치 옆에 설명서 붙이기

  • 예시 코드 : https://github.com/eunjs71/ISC2022-2/tree/main/examples/how_to_use
  • 예시 스케치 보기

게임에 대한 조작 방법을 설명하는 것은 단계 별로 스테이지 시작 전에 스케치 내에서 보여줄 수 있습니다.

조작법이 매우 복잡한 경우, 안내 이미지를 만들어 항상 띄워두는게 좋을 수도 있습니다.

안내 이미지의 크기 만큼 스케치를 넓혀 항상 띄우는 것도 매우 비효율적인 방식일 것입니다.

따라서, HTML, CSS를 조금 활용하여 이미지 파일을 직접 캔버스 옆에 띄워볼 것입니다.

Flexbox로 HTML 요소 정렬하기

  • Flexbox

아직 HTML, CSS에 대해 잘 모르시거나, 깊게 배우지 않은 상태에서 무언가를 해야한다면 "CSS에서 모든 엘리먼트는 display: flex;를 쓰고 시작한다."고 생각하세요. 자세한 설명은 여기서 하지 않겠습니다.

display: flex;가 적용된 엘리먼트는 내부의 요소들의 주축(Main Axis) 방향으로 나열하며, flex-direction 속성을 통해 주축의 방향을 결정할 수 있습니다.

  • flex-direction: row; (기본값): 주축을 수평으로 정함 (왼쪽 → 오른쪽)
  • flex-direction: column;: 주축을 수직으로 정함 (위쪽 → 아래쪽)

따라서 myContainer라는 요소를 만들고, 그 안에 main(스케치가 생성되는 곳), img(이미지)를 넣어 줄 것입니다.

index.html은 다음과 같이 됩니다.

  • 변경 전

            <body>
                <main></main>
            </body>
  • 변경 후

                <body>
                    <div id="myContainer">
                        <main></main>
                        <img src="image/how_to_use.png" />
                    </div>
                </body>

CSS에서 myContainer에는 display: flex; 속성을 붙여줍니다. myContainer라는 id를 가진 요소에 대한 스타일을 지정할 때는 앞에 #을 붙여줍니다. 


            #myContainer {
                display: flex;
                flex-direction: row;
              }

CSS까지 적용하면 나오는 화면은 다음과 같습니다.

이미지가 너무 크므로 조금 줄여보겠습니다.

이미지 태그에 how-to-use라는 id를 붙이고 CSS에서 너비와 높이를 조정합니다. 


            <body>
                <div id="myContainer">
                    <main></main>
                    <img id="how-to-use" src="image/how_to_use.png" />
                </div>
            </body>
        

            #how-to-use {
                width: 300px;
                height: 300px;
              }

이미지의 크기가 알맞게 조정되었습니다.

하지만 두 요소가 화면 위쪽으로 붙어 있는 게 아니라 myContainer의 주축(왼쪽 → 오른쪽)에 수직인 방향을 기준으로 가운데 정렬을 하고 싶습니다.

주축 기준 정렬은 justify-content, 주축에 직각인 방향 정렬은 align-items를 통해 할 수 있습니다. 


            #myContainer {
                display: flex;
                flex-direction: row;
                align-items: center;
              }

여기에서 주축 방향으로 가운데 정렬(justify-content:center;)을 하면 어떻게 될까요? 


            #myContainer {
                display: flex;
                flex-direction: row;
                justify-content: center;
                align-items: center;
              }

window 기준으로 화면 가운데에 오게 됩니다.

myContainer의 높이는 내부 요소들로 결정 되기 때문에, 상하 기준으로는 화면 가운데로 오지 못합니다.

따라서 myContainer가 화면 한 가운데 오게 하고 싶다면, 크기를 화면 만큼 키워주면 됩니다.

고정된 크기는 px 단위를 쓰지만, 화면 크기에 비례한 사이즈를 쓰고 싶으면 vw, vh를 씁니다.

100vw는 창 내부 너비, 100vh는 창 내부 높이입니다. 알맞은 크기에 맞게 사용하시면 됩니다. 


            #myContainer {
                width: 100vw;
                height: 100vh;
                display: flex;
                flex-direction: row;
                justify-content: center;
                align-items: center;
              }

이미지가 너무 붙어 있어 여백을 주고 싶으면 margin을 사용합니다. 여기서는 왼쪽 여백만 있으면 되므로 margin-left를 사용합니다. 


            #how-to-use {
                width: 300px;
                height: 300px;
                margin-left: 40px;
              }
