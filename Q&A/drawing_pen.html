draw() 함수 내에서 그림 그리기

  • 예시 코드 : https://github.com/eunjs71/ISC2022-2/tree/main/examples/drawing_pen
  • 예시 스케치 보기

강의 시간에 mouseX, mouseY, pmouseX, pmouseY, mouseIsPressed를 이용하여 마우스로 그림을 그리는 기능을 구현한 적이 있습니다.

당시에는 배경을 업데이트 하지 않았는데, 배경을 업데이트하는 상황에서 해당 기능을 구현하려니, 배경이 그렸던 line을 지우는 일이 발생합니다.

이 문제를 해결하기 위한 방법 중 하나는 createGraphics()를 이용하는 것입니다.

예를 들어 포토샵을 사용할 때 처럼, canvas내에 여러 개의 레이어가 있다면, 배경은 따로 그리고 내가 펜으로 그리는 레이어는 배경을 업데이트 하지 않고 투명하게 둔 채, 펜으로 그린 자취를 누적시킬 수 있습니다.

그러한 레이어를 createGraphics()을 통해 만들 수 있습니다. 레이어를 생성하고, 그 안에 그림을 그리는 방법은 다음과 같습니다. 


            function setup() {
                createCanvas(400, 400);
                ...
                pg_drawing = createGraphics(width, height);
              }
            function draw_drawing() {
                ...
                if (mouseIsPressed) {
                  pg_drawing.stroke(0);
                  pg_drawing.line(mouseX, mouseY, pmouseX, pmouseY);
                }
                ...
              }

그릴 때 마우스 커서는 cursor(), noCursor() 함수로 보이거나 보이지 않게 할 수 있습니다.

clear() 함수는 캔버스나, 레이어(createGraphics()로 생성한 p5.Render 객체)의 내용을 모두 지우는 데 사용합니다.
